什麼是 MCP？(2 分鐘搞懂)

MCP = Model Context Protocol，Anthropic 在 2024 年底開源的協議。它定義「LLM 怎麼跟外部系統溝通」— 你的網站如果有 MCP server，ChatGPT / Claude / Perplexity 等 AI 就能跟你的後台直接對話：查產品、查價、下單，全程不用客人離開對話框。

簡單講：網站結構化 = SEO / GEO 給搜尋引擎讀；網站 MCP-ready = 給 AI 直接操作，後者是更高一層的整合。

OpenAI / Google / 微軟在 2025 年都已支援 MCP。它不是「會不會普及」的問題，是「你準備好了沒」的問題。

LLM 支援現況主要 AI 引擎都已經支援 MCP。
